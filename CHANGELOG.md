# Changelog

All notable changes to GEOSPH are documented in this file.

## [1.0.0] - 2026-07-15

### Initial Release

First public release of GEOSPH, a Smoothed Particle Hydrodynamics (SPH) framework for computational geomechanics.

### Included Features

- Single-phase solid mechanics simulations
- Elastoplastic constitutive modeling
- Drucker–Prager yield criterion
- Von Mises yield criterion
- Mohr–Coulomb yield criterion
- Particle shifting algorithms
- No-slip and slip-free boundary condition formulations
- Rigid body interaction capabilities

### Future Development

Future releases may include additional capabilities, including:

- Coupled fluid flow–solid deformation formulations
- Advanced critical-state constitutive models
- Additional verification and validation examples
- Expanded documentation and tutorials
